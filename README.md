### Requisitos
* MongoDB
* npm
* NodeJs

### Instalação

Para instalar o programa completo com todas suas dependências, apenas execute o script `setup.sh` com os seguintes comandos no terminal:

```
chmod +x setup.sh
./setup.sh
```
### Execução

Para incializar o programa, execute o seguinte comando:
```
npm start
```
